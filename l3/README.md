## How to run the code

Make sure to activate your virtual environment with the packages tkinter, numpy and pygame installed.

```bash
source .venv/bin/activate
```

* To play against an AI player, run the following command:
```bash
python3 ConnectFour.py ai human
```

* To simulate AI player vs AI player, run the following command:
```bash
python3 ConnectFour.py ai ai
```

* To play against a human player, run the following command:
```bash
python3 ConnectFour.py human human
```

* To play against a random player, run the following command:
```bash
python3 ConnectFour.py random human
```

* To simulate AI player vs random player, run the following command:
```bash
python3 ConnectFour.py ai random
```

* To simulate random player vs random player, run the following command:
```bash
python3 ConnectFour.py random random
```
